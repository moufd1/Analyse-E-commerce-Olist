## 📝 Présentation du Projet
Analyse complète des données de la plateforme e-commerce brésilienne **Olist** (100k commandes). L'objectif est de fournir une vision 360° de la performance commerciale et logistique.

## 🛠️ Compétences Techniques démontrées :
* **Nettoyage de données (ETL) :** Utilisation de Power Query pour traiter les dates, les formats numériques et les valeurs manquantes.
* **Modélisation de données :** Création d'un modèle en étoile (Star Schema) avec tables de faits et dimensions.
* **Calculs DAX complexes :** * Création de KPIs (Chiffre d'Affaires, Panier Moyen).
  * Moyennes mobiles sur 30 jours pour l'analyse de tendance.
  * Segmentation client via la **Matrice RFM** (Récence, Fréquence, Montant).
* **Visualisation de données :** Cartographie choroplèthe (Bing Maps), Entonnoir de conversion, et analyses temporelles.

## 📈 Insights Clés :
* **Concentration géographique :** Plus de 50% du CA est réalisé dans l'État de São Paulo.
* **Logistique :** Un taux de livraison à temps exceptionnel de 89%.
* **Panier Moyen :** Stable autour de 160 R$.
